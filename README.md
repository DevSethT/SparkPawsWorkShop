# 🛠️ SparkPaws Workshop

SparkPaws Workshop is a personal electronics teardown, inventory management, and project tracking application built with React. It was created to organize repair projects, document hardware teardowns, and keep track of reusable electronic components while continuing to expand my full-stack development skills.

This project is currently frontend-focused using Local Storage for persistence, with plans to migrate to a full backend using Node.js, Express, MongoDB, and authentication.

---

## 🌐 Live Demo

**Website:** https://sparkpawsworkshop.com

## 📸 Features

### 📁 Project Management
- Create new teardown projects
- Edit existing projects
- Store descriptions and notes
- Persistent data using Local Storage
- Individual project detail pages

### 📦 Inventory System
- Add electronic components
- Track quantities
- Edit inventory items
- View detailed information for each component
- Inventory persists between sessions

### 🎨 User Experience
- Responsive design
- Blueprint-inspired UI
- React Router navigation
- Local Storage persistence
- Mobile friendly layout

## 🚀 Running Locally

Clone the repository

```bash
git clone https://github.com/DevSethT/SparkPawsWorkShop.git
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
